Query Syntax
http://msdn.microsoft.com/en-us/library/bb308959.aspx#linqoverview_topic5

This:

IEnumerable<string> query = names 
                            .Where(s => s.Length == 5) 
                            .OrderBy(s => s)
                            .Select(s => s.ToUpper());

...is equivalent to this:

IEnumerable<string> query = from s in names 
                            where s.Length == 5
                            orderby s
                            select s.ToUpper();
